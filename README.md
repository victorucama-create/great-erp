# 🚀 Great ERP - Sistema de Gestão Empresarial

Sistema ERP completo para pequenas e médias empresas.

## ✨ Funcionalidades

- ✅ **Multi-tenant** - Várias empresas na mesma plataforma
- ✅ **Gestão de Vendas** - Faturas, clientes
- ✅ **Inventário** - Stock, produtos
- ✅ **Financeiro** - Tesouraria
- ✅ **Super Admin** - Gestão global

## 🚀 Deploy no Render.com

1. **Crie Web Service:**
   - Name: `great-erp`
   - Runtime: `Node`
   - Build Command: `npm install`
   - Start Command: `npm start`

2. **URL:** `https://great-erp.onrender.com`
